## [0.1.3] - 2025-12-21

### Fixed
- 修復 API Key 保存狀態提示的顯示邏輯
- 修復 IP 檢測結果的翻譯問題
- 修復 FlagIcons.vue 的構建錯誤


## [0.1.2] - 2025-12-20

### Changed
- 統一所有應用圖標為 favicon.ico
- 優化 API Key 保存狀態顯示邏輯
- 改進錯誤消息的多語言支持


## [0.1.1] - 2025-12-19

### Added
- 系統托盤功能，支持最小化到托盤
- 單實例鎖定，確保一次只能開啟一個應用實例
- 完整的 IP 檢測和 API 測試功能
- 多語言支持（英文、簡體中文、繁體中文）
- 深色/淺色主題切換，支持系統主題檢測
- 語言選擇器帶國旗圖標
